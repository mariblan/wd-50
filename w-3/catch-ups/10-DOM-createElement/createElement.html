DOM: - EventListener

Definition

The classList property returns the CSS classnames of an element.

Attribute Methods

  • add() - Adds one or more class names to an element. If the specified class already exists, the class will not be added.
  • remove() - Removes one or more class names from an element. Note: Removing a class that does not exist, does NOT throw an error.

Steps

  1. Check the domClassList.css file:
    • There is a class called "hidden".
    • We are going to remove that class to our element with id "dropdown" in the HTML file when hovering over the element with the id "dropdown-toggle".
    • We will add it back when the element is not hovering over the "dropdown" element anymore.
  2. Get the element with the id "dropdown-toggle"
    • Store it in a variabled called "dropdown-toggle"
  3. Get the element with the id "dropdown":
    • Store it into a variable named "dropdown"
  4. Add an event listener to the "dropdown-toggle" element
    • Pass two arguments:
      • The type of event: "mouseover"
      • An anonymous function:
        • Needs no parameter
        • Removes the class "hidden" to the "dropdown" element (classList.remove())
          • You should see how the element appear when hovering ofer the element "dropdown-toggle".
  5. Add an event listener to the "dropdown" element
    • Pass two arguments:
      • The type of event: "mouseout"
      • An anonymous function:
        • No parameter needed
        • Remove the class "hidden" to the "dropdown" element (classList.remove())
          • You should see the dropdown appearing again when you stop hovering over the "dropdown".
