Local Storage - getItem()

Definition

The getItem() method returns value of the specified Storage Object item.

The value will always be returned as a string, so if what was saved was an array or an object, it will need to be parsed.

The parse method is used to transform the stringify value you are recieving into an object.

How to set an item

  • Get an item: localStorage.getItem('key')
  • Getting the value when it is not a string:
    • JSON.parse(localStorage.getItem('key'))

Steps

  1. Get the item with the key myName from localStorage:
    • Log the value into your console
  2. Get the item with the key strArray from localStorage:
    • Use the parse method to transform the stringify value you are recieving into an object.
    • Log the value into your console
  3. Get the item with the key objArray from localStorage:
    • Use the parse method to transform the stringify value you are recieving into an object.
    • Log the value into your console

How to check localStorage?

Open the developer tools in your browser and go to the Application tab. You will see the localStorage object and the items you have saved.
