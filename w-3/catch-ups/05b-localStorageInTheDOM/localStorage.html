Local Storage

Definition

Local storage is a read-only property of the window interface that allows you to access a storage object for the Document's origin.

Local Storage data is saved accross sessions, so it is a good place to store data that you want to persist.

How to use localStorage

  • Set an item: localStorage.setItem('key', 'value')
  • Get an item: localStorage.getItem('key')
  • Remove an item: localStorage.removeItem('key')
  • Clear all items: localStorage.clear()

Steps

  1. Add an item to your localStorage:
    • Name the key string
    • Pass your name as a value
  2. Get the item that you saved in localStorage:
    • Store it in a variable called "myName"
  3. Display the value you stored in the "myName" varible in the span with id "my-name":
    • Get the span with DOM manipulation.
    • Change the innerText of the span to be the value of your variable.

How to check localStorage?

Open the developer tools in your browser and go to the Application tab. You will see the localStorage object and the items you have saved.
