Local Storage

Definition

Local storage is a read-only property of the window interface that allows you to access a storage object for the Document's origin.

Local Storage data is saved accross sessions, so it is a good place to store data that you want to persist.

How to use localStorage

  • Set an item: localStorage.setItem('key', 'value')
  • Get an item: localStorage.getItem('key')
  • Remove an item: localStorage.removeItem('key')
  • Clear all items: localStorage.clear()
