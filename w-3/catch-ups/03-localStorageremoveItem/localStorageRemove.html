Local Storage: - removeItem()

Definition

The removeItem() method removes the specified Storage Object item.

How to remove an item

  • Remove an item: localStorage.removeItem('key')

Steps

  1. Remove the item with the key myName from localStorage.
  2. Remove the item with the key strArray from localStorage.
  3. Remove the item with the key objArray from localStorage.

How to check localStorage?

Open the developer tools in your browser and go to the Application tab. You will see the localStorage object and the items you have saved.
