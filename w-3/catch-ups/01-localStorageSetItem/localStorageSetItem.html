Local Storage - setItem()

Definition

Local storage is a read-only property of the window interface that allows you to access a storage object for the Document's origin.

Local Storage data is saved accross sessions, so it is a good place to store data that you want to persist.

The setItem() method sets the value of the specified Storage Object item.

The value will be stored as a string, so if the value that wants to be saved is not a string, it will need to be stringified.

How to set an item

  • Set an item: localStorage.setItem('key', 'value')
  • Setting the value when it is not a string: JSON.stringify(value)

Steps

  1. Add an item with the value of a string to your localStorage:
    • Name the key string
    • Pass your name as a value
  2. Add a second item with the value of an array to your localStorage:
    • Name the key strArray.
    • Pass and array of strings as its value.
  3. Add a third item with the value of an array to your localStorage
    • Name the key objArray
    • Pass the value of an array of objects.

How to check localStorage?

Open the developer tools in your browser and go to the Application tab. You will see the localStorage object and the items you have saved.
