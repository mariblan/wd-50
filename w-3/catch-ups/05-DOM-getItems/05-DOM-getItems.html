DOM: - get items

Definition

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

Getting One element

  • Get an element by id: document.getElementById('id')
  • Get elements by query selector: document.querySelector('selector')

Getting Multiple elements

  • Get elements by class name: document.getElementsByClassName('class')
  • Get elements by tag name: document.getElementsByTagName('tag')
  • Get elements by query selector all: document.querySelectorAll('selector')
