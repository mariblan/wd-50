DOM: - attribute methods

Definition

The getAttribute() method returns the value of an element's attribute.

The hasAttribute() method returns true if the attribute exists, otherwise false.

The setAttribute() method sets the value of an attribute on the specified element.

The removeAttribute() method removes an attribute from an element.

Attribute Methods

  • Get the value of anattribute-> element.getAttribute("attributeName")
  • Check if an attribute exists in an element-> element.hasAttribute("attributeName")
  • Set the value of an attribute-> element.getAttribute("attributeName", "value")
  • Remove an attribute from an element-> element.removeAttribute("attributeName")
